# News Reader

### Overview:
This demo application is built using **ASP**.NET Core 3 Web API and Angular 8 to use data from the [Hacker News API](https://github.com/HackerNews/API)

Other .Net Libraries Used:
- Polly
- Newtonsoft.Json
- FluentAssertions
- Moq
- XUnit
